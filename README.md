# 网站链接整理记录

本仓库用于自动生成第三方网站入口整理页面。

## 自动发布设置

- 发布时间：北京时间每天18:00
- 每日生成：20篇
- 每篇域名：6个
- 自动汇总：`daily-links/`
- 最近链接：`latest-links.txt`

## 使用说明

- `domains.txt`：维护域名列表
- `config.json`：控制每天篇数和每篇域名数量
- `posts/`：自动生成的文章
- `daily-links/`：每天的链接汇总
- `latest-links.txt`：最近一次发布的全部链接
